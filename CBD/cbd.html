Presentación de la Asignatura

Introducción

Bienvenido a la asignatura Estrategias y Herramientas de Computación Big Data en la Nube (CBD), del Máster Universitario en Computación en la Nube y de Altas Prestaciones (MUCNAP) de la Universitat Politècnica de València. Este seminario ofrece a los alumnos experiencia en la utilización de modelos de programación orientados a Cloud. En primer lugar se abordan los modelos de programación orientados a cómputo intensivo y se aborda el despliegue de clusters virtuales elásticos en la nube. A continuación se aborda el modelo MapReduce para el análisis de grandes volúmenes de datos, implementado mediante Apache Hadoop. Posteriormente, se abordan herramientas de nivel más alto para interactuar con un cluster Hadoop (como Apache Hue) y herramientas para consultar datasets mediante lenguaje similar a SQL (Apache Hive). También se aborda el funcionamiento de Apache Spark.

Tema 1. Computación Intensiva en la Nube

Este tema se centra en los modelos de programación orientados a cálculo intensivo, específicamente sobre aquellos más relacionados con las tecnologías Cloud. Se introducen algunas herramientas para el despliegue de clusters virtuales en la nube y se realiza una práctica de despliegue de clusters virtuales elásticos sobre Amazon Web Services usando la herramienta AWS ParallelCluster.

Tema 2. Procesado de Big Data en la Nube

Este tema aborda el modelo de programación MapReduce para el análisis de conjuntos de datos masivos así como su implementación mediante Apache Hadoop. Se realiza un práctica de uso de Hadoop sobre un cluster dinámicamente desplegado en Amazon Web Services para procesar la información de diversos conjuntos de datos. También se aborda Amazon EMR para auto-aprovisionar clusters Hadoop sobre AWS y se utiliza Apache Hue para interactuar con Apache Hadoop de forma gráfica. Finalmente, se utiliza Apache Hive para realizar consultas SQL a datasets almacenados en el sistema de archivos distribuido de Hadoop.
